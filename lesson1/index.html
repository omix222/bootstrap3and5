Bootstrap 3
BS3 の特徴： Float ベース。最小は col-xs-*。ブレークポイントは xs / sm / md / lg の4段階。

1. 最小ブレークポイント（xs）

col-xs-12（全幅）
col-xs-6
col-xs-6
<div class="col-xs-12"> <div class="col-xs-6">

2. 3カラムレイアウト

col-md-4
col-md-4
col-md-4
<div class="row"> <div class="col-md-4">...</div> <div class="col-md-4">...</div> <div class="col-md-4">...</div> </div>

3. オフセット

col-md-4 col-md-offset-4
<div class="col-md-4 col-md-offset-4">

ブレークポイント一覧（BS3）

xs(<768px) / sm(≥768px) / md(≥992px) / lg(≥1200px)
Bootstrap 5
BS5 の変更点： Flexbox ベースに刷新。col-xs-* は廃止され col-* に。 xxl（≥1400px）ブレークポイントが追加。

1. 最小ブレークポイント（xs 廃止 → col）

col-12（全幅）
col-6
col-6
<div class="col-12"> <!-- xs は不要 --> <div class="col-6">

2. 3カラムレイアウト（gutter 追加）

col-md-4
col-md-4
col-md-4
<div class="row g-3"> <!-- g-* で gutter 指定 --> <div class="col-md-4">...</div> <div class="col-md-4">...</div> <div class="col-md-4">...</div> </div>

3. オフセット（クラス名変更）

col-md-4 offset-md-4
<div class="col-md-4 offset-md-4"> <!-- BS3: col-md-offset-4 → BS5: offset-md-4 -->

ブレークポイント一覧（BS5）

col(<576px) / sm(≥576px) / md(≥768px) / lg(≥992px) / xl(≥1200px) / xxl(≥1400px)

4. Auto-layout Columns（BS5 新機能）

col

col

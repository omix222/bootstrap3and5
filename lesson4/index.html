Bootstrap 3
BS3 の特徴： form-group で各フィールドをラップ。インラインフォームは form-inline。 バリデーション状態は has-error / has-success を親に付与。

1. 基本フォーム（form-group）

入力してください。
<form> <div class="form-group"> <label for="email">メール</label> <input type="email" class="form-control" id="email" placeholder="..."> <span class="help-block">補足</span> </div> <div class="form-group"> <label>パスワード</label> <input type="password" class="form-control"> </div> <button class="btn btn-primary">ログイン</button> </form>

2. バリデーション表示（has-error / has-success）

エラーメッセージ
<div class="form-group has-error"> <label>エラー</label> <input class="form-control"> <span class="help-block">エラー文</span> </div> <div class="form-group has-success"> <!-- 親要素にクラスを付与するスタイル --> </div>

3. チェックボックス・ラジオ

<div class="checkbox"> <label><input type="checkbox"> A</label> </div> <div class="radio"> <label><input type="radio"> 1</label> </div>
Bootstrap 5
BS5 の変更点： form-group は廃止 → mb-3 などのスペーシングで代替。 バリデーションは is-invalid / is-valid を直接 input に付与。 フローティングラベルが新登場。

1. 基本フォーム（mb-3 で代替）

入力してください。
<form> <div class="mb-3"> <!-- form-group 廃止 --> <label for="email" class="form-label">メール</label> <input type="email" class="form-control" id="email" placeholder="..."> <div class="form-text">補足</div> </div> <div class="mb-3"> <label class="form-label">パスワード</label> <input type="password" class="form-control"> </div> <button class="btn btn-primary">ログイン</button> </form>

2. バリデーション（is-invalid / is-valid）

エラーメッセージ
OK
<input class="form-control is-invalid"> <div class="invalid-feedback">エラー文</div> <input class="form-control is-valid"> <div class="valid-feedback">OK</div> <!-- input に直接クラスを付与（親要素不要） -->

3. チェックボックス・ラジオ（form-check 統一）

<div class="form-check"> <input class="form-check-input" type="checkbox" id="a"> <label class="form-check-label" for="a">A</label> </div> <div class="form-check"> <input class="form-check-input" type="radio" id="r1"> <label class="form-check-label" for="r1">1</label> </div>

4. フローティングラベル（BS5 新機能）

<!-- BS5 新機能：フローティングラベル --> <div class="form-floating mb-3"> <input type="email" class="form-control" id="floatEmail" placeholder="name@example.com"> <label for="floatEmail">メール</label> </div> <!-- placeholder 属性が必須（ラベル位置制御に使用） -->

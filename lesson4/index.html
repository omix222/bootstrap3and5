Bootstrap 3
form-group でラップ。バリデーション状態は親に has-error/has-success。jQuery 必須。

1. 基本フォーム（form-group）

入力してください。
<div class="form-group"> <label>メール</label> <input class="form-control"> <span class="help-block">補足</span> </div>

2. バリデーション（親要素に has-error）

エラーメッセージ
<div class="form-group has-error"> <label>エラー</label> <!-- 親要素に付与 --> <input class="form-control"> <span class="help-block">メッセージ</span> </div> <div class="form-group has-success">

3. チェックボックス・ラジオ

<div class="checkbox"> <label><input type="checkbox"> A</label> </div> <div class="radio"> <label><input type="radio"> 1</label> </div>

4. インラインフォーム（form-inline）

<form class="form-inline"> <div class="form-group"> <label>名前</label> <input class="form-control"> </div> <button class="btn btn-primary">送信</button> </form>

5. Input Group（BS3 addon）

@
<div class="input-group"> <span class="input-group-addon">@</span> <input class="form-control"> </div> <div class="input-group"> <input class="form-control"> <div class="input-group-btn"> <button class="btn btn-default">検索</button> </div> </div>

6. Range / Switch（custom-range / custom-switch）

<input type="range" class="custom-range"> <div class="custom-control custom-switch"> <input type="checkbox" class="custom-control-input" id="sw"> <label class="custom-control-label" for="sw"> スイッチ </label> </div>

7. ファイル入力（custom-file）

ファイルを選択してください...
<div class="custom-file"> <input type="file" class="custom-file-input" id="file"> <label class="custom-file-label" for="file"> ファイルを選択 </label> </div>

8. フォームサイズ

<input class="form-control input-lg"> <input class="form-control"> <input class="form-control input-sm">
Bootstrap 5
form-group 廃止→mb-3。バリデーションは input 自身に。form-check 統一。フローティングラベル新登場。

1. 基本フォーム（mb-3 で代替）

入力してください。
<div class="mb-3"> <!-- form-group 廃止 --> <label class="form-label">メール</label> <input class="form-control"> <div class="form-text">補足</div> </div>

2. バリデーション（input に直接 is-invalid/is-valid）

エラーメッセージ
OK！
<input class="form-control is-invalid"> <div class="invalid-feedback">エラー文</div> <input class="form-control is-valid"> <div class="valid-feedback">OK</div> <!-- input に直接付与（親要素不要）-->

3. チェックボックス・ラジオ（form-check 統一）

<div class="form-check"> <input class="form-check-input" type="checkbox" id="a"> <label class="form-check-label" for="a"> A </label> </div> <!-- radio も同じ form-check クラスを使用 -->

4. インラインフォーム（form-inline 廃止 → row + col-auto）

<!-- form-inline 廃止 → row + col-auto --> <form class="row g-3 align-items-end"> <div class="col-auto"> <label class="form-label">名前</label> <input class="form-control"> </div> <div class="col-auto"> <button class="btn btn-primary">送信</button> </div> </form>

5. Input Group（addon → input-group-text）

@
¥ .00
<div class="input-group"> <span class="input-group-text">@</span> <input class="form-control"> </div> <!-- input-group-addon → input-group-text --> <!-- input-group-btn → 直接 button を並べる --> <!-- 両側テキスト --> <div class="input-group"> <span class="input-group-text">¥</span> <input class="form-control"> <span class="input-group-text">.00</span> </div>

6. Range / Switch（form-range / form-check form-switch）

<!-- custom-range → form-range --> <input type="range" class="form-range"> <!-- custom-switch → form-check form-switch --> <div class="form-check form-switch"> <input class="form-check-input" type="checkbox" id="sw"> <label class="form-check-label" for="sw"> スイッチ </label> </div>

7. ファイル入力（custom-file 廃止）

<!-- custom-file 廃止 → form-control type="file" --> <input class="form-control" type="file" id="file"> <!-- 複数選択 --> <input class="form-control" type="file" multiple>

8. フォームサイズ（クラス名変更）

<!-- input-lg/sm → form-control-lg/sm --> <input class="form-control form-control-lg"> <input class="form-control form-control-sm"> <!-- select: form-control → form-select --> <select class="form-select"> <select class="form-select form-select-lg"> <select class="form-select form-select-sm">

9. フローティングラベル（BS5 新機能）

<!-- BS5 新機能：フローティングラベル --> <!-- placeholder 属性が必須（位置制御に使用） --> <div class="form-floating mb-3"> <input type="email" class="form-control" id="email" placeholder="dummy"> <label for="email">メールアドレス</label> </div> <!-- select / textarea にも対応 --> <div class="form-floating"> <select class="form-select" id="sel">...</select> <label for="sel">カテゴリ</label> </div>

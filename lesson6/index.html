Lesson 6：ユーティリティクラスの変更（RTL 対応）

  1. ガイドトップ
  2. Lesson 6
Bootstrap 3
left/right を直接クラス名に含む。RTL（右から左）言語を考慮していない設計。

1. スペーシング（m- / p- 形式）

<!-- Margin --> <div class="ml-3">左マージン</div> <div class="mr-3">右マージン</div> <div class="mt-3">上マージン</div> <div class="mb-3">下マージン</div> <div class="mx-3">左右マージン</div> <div class="my-3">上下マージン</div> <div class="m-3">全方向マージン</div> <!-- Padding --> <div class="pl-3">左パディング</div> <div class="pr-3">右パディング</div>

2. テキスト整列

text-left（左揃え）
text-center（中央揃え）
text-right（右揃え）
<div class="text-left">左揃え</div> <div class="text-center">中央揃え</div> <div class="text-right">右揃え</div> <!-- レスポンシブ版 --> <div class="text-md-right">MD以上で右揃え</div>

3. Float

pull-left
pull-right
<div class="pull-left">左に float</div> <div class="pull-right">右に float</div> <div class="clearfix"></div>

4. Border / Rounded（left/right を直接指定）

<div class="border-left">左ボーダー</div> <div class="border-right">右ボーダー</div> <div class="rounded-left">左丸角</div> <div class="rounded-right">右丸角</div>

5. 表示・非表示

<!-- 個別制御 --> <div class="show">表示</div> <div class="hide">非表示</div> <div class="invisible">不可視（スペース確保）</div> <span class="sr-only">スクリーンリーダーのみ</span> <!-- BS4 以降 sr-only は Bootstrap 本体から分離 -->

6. カラーユーティリティ

<!-- テキスト色 --> <p class="text-muted">グレー文字</p> <p class="text-primary">青文字</p> <p class="text-success">緑文字</p> <p class="text-danger">赤文字</p> <p class="text-warning">黄文字</p> <!-- 背景色（BS3 は bg-* が少ない） --> <div class="bg-primary">青背景</div> <div class="bg-success">緑背景</div>

7. ボタンブロック（btn-block）

<button class="btn btn-primary btn-block"> 全幅ボタン </button>

8. 閉じるボタン

<button type="button" class="close"> <span aria-hidden="true">&times;</span> </button>

9. Embed レスポンシブ

<div class="embed-responsive embed-responsive-16by9"> <iframe class="embed-responsive-item" src="..."></iframe> </div>
Bootstrap 5
RTL（右→左）言語に対応するため left/right → start/end に統一。ms/me/ps/pe などが新クラス名。

1. スペーシング（ms/me/ps/pe に変更）

ms-3（左マージン）
me-3（右マージン）
ps-3（左パディング）
pe-3（右パディング）
<!-- ml-* / mr-* → ms-* / me-* --> <div class="ms-3">左マージン（margin-start）</div> <div class="me-3">右マージン（margin-end）</div> <div class="mx-3">左右マージン（変更なし）</div> <div class="mt-3">上マージン（変更なし）</div> <!-- pl-* / pr-* → ps-* / pe-* --> <div class="ps-3">左パディング（padding-start）</div> <div class="pe-3">右パディング（padding-end）</div>

2. テキスト整列（start / end）

text-start（左揃え）
text-center（中央揃え）
text-end（右揃え）
<!-- text-left/right → text-start/end --> <div class="text-start">左揃え</div> <div class="text-center">中央揃え（変更なし）</div> <div class="text-end">右揃え</div> <!-- レスポンシブ版 --> <div class="text-md-end">MD以上で右揃え</div>

3. Float（float-start / float-end）

float-start
float-end
<!-- pull-left/right 廃止 → float-start/end --> <div class="float-start">左に float</div> <div class="float-end">右に float</div> <div class="clearfix"></div> <!-- clearfix は存続 --> <!-- レスポンシブ版 --> <div class="float-md-end">MD以上で右 float</div>

4. Border / Rounded（start / end）

border-start
border-end
rounded-start
rounded-end
<!-- border-left/right → border-start/end --> <div class="border-start">左ボーダー</div> <div class="border-end">右ボーダー</div> <!-- rounded-left/right → rounded-start/end --> <div class="rounded-start">左丸角</div> <div class="rounded-end">右丸角</div> <!-- ボーダー色・太さも指定可 --> <div class="border-start border-primary border-3">

5. 表示・非表示（d-* / visually-hidden）

d-none d-md-block（MD以上で表示）
このテキストはスクリーンリーダーのみ（visually-hidden）
invisible（不可視だがスペースを占有）
<!-- show/hide 廃止 → d-* ユーティリティ --> <div class="d-none">非表示</div> <div class="d-block">ブロック表示</div> <div class="d-flex">Flex 表示</div> <!-- sr-only → visually-hidden --> <span class="visually-hidden"> スクリーンリーダーのみ </span> <!-- invisible は変更なし --> <div class="invisible">不可視（スペース確保）</div>

6. カラーユーティリティ（BS5 で大幅拡充）

primary secondary success danger warning info light dark
<!-- text-* カラークラス（ほぼ変更なし） --> <p class="text-primary">青文字</p> <p class="text-muted">グレー文字（変更なし）</p> <!-- bg-* カラークラス（BS5 で拡充） --> <div class="bg-primary"> <div class="bg-secondary"> <!-- text-bg-* （BS5.3 新機能: 背景+文字色セット） --> <div class="text-bg-primary">背景+文字一括</div> <!-- CSS カスタムプロパティで上書き可能 --> :root { --bs-primary: #your-color; --bs-primary-rgb: r, g, b; }

7. ボタンブロック（btn-block 廃止 → d-grid）

<!-- btn-block 廃止 → d-grid --> <div class="d-grid gap-2"> <button class="btn btn-primary">全幅ボタン</button> </div> <!-- レスポンシブ：MD以上でインライン --> <div class="d-grid d-md-flex gap-2"> <button class="btn btn-primary">ボタン</button> </div>

8. 閉じるボタン（btn-close に統一）

<!-- close + ×span → btn-close --> <button type="button" class="btn-close" aria-label="Close"></button> <!-- 白抜き版（暗い背景用） --> <button class="btn-close btn-close-white"></button>

9. Embed レスポンシブ（ratio に変更）

16:9
<!-- embed-responsive → ratio --> <div class="ratio ratio-16x9"> <iframe src="..." allowfullscreen></iframe> </div> <!-- 利用可能な比率 --> <div class="ratio ratio-1x1">正方形</div> <div class="ratio ratio-4x3">4:3</div> <div class="ratio ratio-16x9">16:9</div> <div class="ratio ratio-21x9">映画サイズ</div>

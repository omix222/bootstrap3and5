Bootstrap 5
badge と label を badge に統合。alert は btn-close + data-bs-dismiss に。Spinner が新登場。

1. Badge（label と統合・bg-* で色指定）

通知数: 4 メール: 12 エラー: !

Primary Success Warning Info Dark

ピル型: Primary Danger

ボタン内バッジ:

<!-- badge + bg-* で色指定（label 廃止・統合） --> <span class="badge bg-primary">Primary</span> <span class="badge bg-secondary">Secondary</span> <span class="badge bg-success">Success</span> <span class="badge bg-danger">Danger</span> <!-- ピル型（label-pill → rounded-pill） --> <span class="badge rounded-pill bg-primary">ピル</span> <!-- ボタン内バッジ --> <button class="btn btn-primary"> 受信 <span class="badge bg-danger">9</span> </button>

2. Alert（btn-close + data-bs-dismiss）

成功！ 操作が完了しました。
エラー！ 問題が発生しました。
情報メッセージ
警告メッセージ
<div class="alert alert-success alert-dismissible fade show" role="alert"> <strong>成功！</strong> メッセージ <button type="button" class="btn-close" data-bs-dismiss="alert"></button> </div> <!-- jQuery 不要。fade show でアニメーション -->

3. Button（btn-xs 廃止・btn-block 廃止）

<button class="btn btn-primary btn-lg">大</button> <button class="btn btn-primary">通常</button> <button class="btn btn-primary btn-sm">小</button> <!-- btn-xs は廃止 --> <!-- outline バリアント（BS3と同様） --> <button class="btn btn-outline-primary"> <!-- btn-block 廃止 → d-grid --> <div class="d-grid"> <button class="btn btn-primary">全幅</button> </div>

4. Progress Bar（aria 属性の変更）

60%
80%
30%
<div class="progress"> <div class="progress-bar bg-success" role="progressbar" style="width:80%" aria-label="80%" aria-valuenow="80" aria-valuemin="0" aria-valuemax="100"> 80% </div> </div> <!-- アニメーションストライプ --> <div class="progress-bar progress-bar-striped progress-bar-animated">

5. List Group（active / disabled を a タグで表現）

  • アクティブ項目 14
  • 通常の項目
  • Success カラー
  • Danger カラー
  • 無効な項目
リンク（active） リンク
<ul class="list-group"> <li class="list-group-item active d-flex justify-content-between"> アクティブ <span class="badge bg-primary rounded-pill">14</span> </li> <li class="list-group-item list-group-item-success"> カラーバリアント </li> <li class="list-group-item disabled" aria-disabled="true">無効</li> </ul> <!-- ボタン/リンク型 --> <div class="list-group"> <a class="list-group-item list-group-item-action"> クリッカブル </a> </div>

6. Breadcrumb（active を li ではなく aria-current で）

  1. ホーム
  2. カテゴリ
  3. 現在のページ
<nav aria-label="breadcrumb"> <ol class="breadcrumb"> <li class="breadcrumb-item"> <a href="#">ホーム</a> </li> <li class="breadcrumb-item"> <a href="#">カテゴリ</a> </li> <li class="breadcrumb-item active" aria-current="page"> 現在のページ </li> </ol> </nav>

7. Spinner（BS5 新機能）

Loading...
<!-- BS5 新機能：Spinner コンポーネント --> <!-- 回転型 --> <div class="spinner-border text-primary" role="status"> <span class="visually-hidden">Loading...</span> </div> <!-- 拡大型 --> <div class="spinner-grow text-success" role="status"></div> <!-- 小サイズ --> <div class="spinner-border spinner-border-sm"></div> <!-- ボタン内に埋め込み --> <button class="btn btn-primary" disabled> <span class="spinner-border spinner-border-sm"></span> 読み込み中... </button>

8. Placeholder（ローディングスケルトン）BS5 新機能

Bootstrap 3
jQuery 1.x/2.x/3.x が必須。data-toggle/data-target/data-dismiss で制御。

1. 読み込み（jQuery 必須）

<!-- head --> <link rel="stylesheet" href="bootstrap.min.css"> <!-- body 末尾（順番厳守） --> <script src="jquery.min.js"></script> <!-- jQuery 1.x / 2.x / 3.x が必須 --> <script src="bootstrap.min.js"></script> <!-- または --> <script src="popper.min.js"></script> <script src="bootstrap.min.js"></script>

2. モーダル（data-toggle / data-dismiss）

<button data-toggle="modal" data-target="#myModal"> 開く </button> <div class="modal fade" id="myModal"> <div class="modal-dialog"> <div class="modal-content"> <div class="modal-header"> <button class="close" data-dismiss="modal"> <span>&times;</span> </button> <h4 class="modal-title">タイトル</h4> </div> <div class="modal-body">本文</div> <div class="modal-footer"> <button data-dismiss="modal">閉じる</button> </div> </div> </div> </div>

3. アコーディオン（panel-group + data-toggle）

<div class="panel-group" id="acc"> <div class="panel panel-default"> <div class="panel-heading"> <a data-toggle="collapse" data-parent="#acc" href="#c1">項目 1</a> </div> <div id="c1" class="panel-collapse collapse in"> <div class="panel-body">内容</div> </div> </div> </div>

4. Tooltip（jQuery プラグイン）

<button data-toggle="tooltip" title="ツールチップ"> ホバー </button> <!-- jQuery で初期化 --> $('[data-toggle="tooltip"]').tooltip()

5. Popover（jQuery プラグイン）

<button data-toggle="popover" data-content="内容" title="タイトル"> クリック </button> $('[data-toggle="popover"]').popover()

6. jQuery プラグイン API

<!-- モーダル --> $('#myModal').modal('show') $('#myModal').modal('hide') $('#myModal').modal('toggle') $('#myModal').on('shown.bs.modal', fn) <!-- タブ --> $('#tab a').on('click', function(e) { e.preventDefault() $(this).tab('show') }) <!-- Collapse --> $('#collapse').collapse('show') $('#collapse').collapse('hide')

7. Carousel（data-ride / data-slide）

<div class="carousel slide" id="car" data-ride="carousel" data-interval="3000"> <ol class="carousel-indicators"> <li data-target="#car" data-slide-to="0" class="active"></li> </ol> <a class="left carousel-control" href="#car" data-slide="prev"> <span class="glyphicon glyphicon-chevron-left"></span> </a> </div> <!-- Glyphicons は BS4 以降廃止 -->

8. Toast（BS3 には存在しない）

Bootstrap 5
jQuery 不要。Vanilla JS のみ。data-bs-* に統一。bootstrap.bundle.min.js 1ファイルで完結。

1. 読み込み（1ファイルのみ）

<!-- head --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"> <!-- body 末尾 --> <!-- jQuery 不要！bundle に Popper.js 内包済み --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

2. モーダル（クリックして確認）

Bootstrap 5 モーダル

jQuery なしで動作しています。data-bs-toggle / data-bs-dismiss で制御。
<button data-bs-toggle="modal" data-bs-target="#modal5">開く</button> <div class="modal fade" id="modal5"> <div class="modal-dialog"> <div class="modal-content"> <div class="modal-header"> <h1 class="modal-title fs-5">タイトル</h1> <button class="btn-close" data-bs-dismiss="modal"></button> </div> </div> </div> </div>

3. アコーディオン（クリックして確認）

項目 1 の内容。accordion-item / accordion-button クラスが刷新。

項目 2 の内容。
<div class="accordion" id="acc"> <div class="accordion-item"> <h2 class="accordion-header"> <button class="accordion-button" data-bs-toggle="collapse" data-bs-target="#a1" data-bs-parent="#acc">項目1</button> </h2> <div id="a1" class="accordion-collapse collapse show"> <div class="accordion-body">内容</div> </div> </div> </div> <!-- "collapse in" → "collapse show" --> <!-- data-parent → data-bs-parent -->

4. Tooltip（ホバーして確認）

<button data-bs-toggle="tooltip" data-bs-placement="top" title="ツールチップ">ホバー</button> <!-- JS で初期化が必要（data-bs-toggle="tooltip" だけでは動かない） --> <script> document.querySelectorAll('[data-bs-toggle="tooltip"]') .forEach(el => new bootstrap.Tooltip(el)) </script>

5. Popover（クリックして確認）

<button data-bs-toggle="popover" data-bs-title="タイトル" data-bs-content="内容"> クリック </button> <!-- JS で初期化が必要 --> <script> document.querySelectorAll('[data-bs-toggle="popover"]') .forEach(el => new bootstrap.Popover(el)) </script> <!-- data-trigger → data-bs-trigger --> <!-- data-content → data-bs-content -->

6. Toast（BS5 新機能）

通知 今
Bootstrap 5 の Toast 通知です。jQuery 不要で動作します。
<!-- BS5 新機能：Toast 通知 --> <div class="toast-container position-fixed bottom-0 end-0 p-3"> <div id="myToast" class="toast" role="alert"> <div class="toast-header"> <strong class="me-auto">通知</strong> <small>今</small> <button class="btn-close" data-bs-dismiss="toast"></button> </div> <div class="toast-body">メッセージ本文</div> </div> </div> <!-- JS で表示 --> <script> const toast = new bootstrap.Toast('#myToast') toast.show() </script>

9. Vanilla JS API

<!-- Vanilla JS クラス API --> const modal = new bootstrap.Modal('#modal5') modal.show() modal.hide() modal.toggle() <!-- 既存インスタンスの取得 --> const inst = bootstrap.Modal.getInstance('#modal5') <!-- イベントリスナー --> document.querySelector('#modal5') .addEventListener('shown.bs.modal', () => { // 表示アニメーション完了後 }) .addEventListener('hidden.bs.modal', () => { // 非表示完了後 }) <!-- Collapse --> const col = new bootstrap.Collapse('#a1') col.show() / col.hide() / col.toggle()

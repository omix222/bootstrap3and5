Bootstrap 3
BS3 の問題点： モーダル・コラプス・ドロップダウン・Tooltip など、ほぼすべてに jQuery が必須。 data-toggle / data-target を使用。

1. 必要な読み込みファイル（BS3）

<!-- head --> <link rel="stylesheet" href="bootstrap.min.css"> <!-- body 末尾（順番が重要！） --> <script src="jquery.min.js"></script> <!-- jQuery 1.x / 2.x / 3.x が必須 --> <script src="bootstrap.min.js"></script>

2. モーダル（data-toggle）

※ 動作デモは右ペイン（BS5）を参照

<!-- トリガーボタン --> <button type="button" data-toggle="modal" data-target="#myModal"> モーダルを開く </button> <!-- モーダル本体 --> <div class="modal fade" id="myModal"> <div class="modal-dialog"> <div class="modal-content"> <div class="modal-header"> <button type="button" class="close" data-dismiss="modal"> <span>&times;</span> </button> <h4 class="modal-title">タイトル</h4> </div> <div class="modal-body">本文</div> <div class="modal-footer"> <button data-dismiss="modal">閉じる</button> </div> </div> </div> </div>

3. アコーディオン（data-toggle）

<div class="panel-group" id="accordion"> <div class="panel panel-default"> <div class="panel-heading"> <a data-toggle="collapse" data-parent="#accordion" href="#collapseOne"> 項目 1 </a> </div> <div id="collapseOne" class="panel-collapse collapse in"> <div class="panel-body">内容</div> </div> </div> </div>

4. JS API（jQuery プラグイン）

<!-- jQuery で制御（BS3） --> $('#myModal').modal('show') $('#myModal').modal('hide') $('#myModal').on('shown.bs.modal', function () { // イベント処理 }) <!-- Tooltip の初期化も jQuery 必須 --> $('[data-toggle="tooltip"]').tooltip()
Bootstrap 5
BS5 の変更点： jQuery を完全に排除。Vanilla JS のみで動作。 data-toggle / data-target → data-bs-toggle / data-bs-target に変更。 data-dismiss → data-bs-dismiss。

1. 必要な読み込みファイル（BS5）

<!-- head --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"> <!-- body 末尾 --> <!-- jQuery 不要！ --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"> </script> <!-- bundle には Popper.js が内包済み -->

2. モーダル（実際にクリックして確認）

<!-- トリガーボタン --> <button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#bs5Modal"> モーダルを開く </button> <!-- モーダル本体 --> <div class="modal fade" id="bs5Modal" tabindex="-1"> <div class="modal-dialog"> <div class="modal-content"> <div class="modal-header"> <h1 class="modal-title fs-5">タイトル</h1> <button type="button" class="btn-close" data-bs-dismiss="modal"> </button> </div> <div class="modal-body">本文</div> <div class="modal-footer"> <button class="btn btn-secondary" data-bs-dismiss="modal"> 閉じる </button> </div> </div> </div> </div>

Bootstrap 5 モーダル

jQuery なしで動いています！
data-bs-toggle="modal" と data-bs-target だけで動作します。

3. アコーディオン（実際にクリックして確認）

項目 1 の内容です。jQuery 不要で動きます。

項目 2 の内容。accordion-item / accordion-button などクラスが刷新。
<div class="accordion" id="myAccordion"> <div class="accordion-item"> <h2 class="accordion-header"> <button class="accordion-button" data-bs-toggle="collapse" data-bs-target="#acc1"> 項目 1 </button> </h2> <div id="acc1" class="accordion-collapse collapse show" data-bs-parent="#myAccordion"> <div class="accordion-body">内容</div> </div> </div> </div>

4. JS API（Vanilla JS）

上のモーダル (#bs5Modal) を JS で制御します

<!-- Vanilla JS で制御（BS5） --> const modal = new bootstrap.Modal('#bs5Modal') modal.show() modal.hide() <!-- イベントリスナー --> document.getElementById('bs5Modal') .addEventListener('shown.bs.modal', () => { // 表示後の処理 }) <!-- Tooltip の初期化（まとめて） --> document.querySelectorAll('[data-bs-toggle="tooltip"]') .forEach(el => new bootstrap.Tooltip(el))

5. Tooltip（JS 初期化が必要）

 
<button data-bs-toggle="tooltip" data-bs-placement="top" title="ツールチップ"> ホバー </button> <!-- Tooltip は JS 初期化が必要 --> <script> document.querySelectorAll('[data-bs-toggle="tooltip"]') .forEach(el => new bootstrap.Tooltip(el)) </script>

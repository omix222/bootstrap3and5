# Bootstrap 5.3 Ultimate Showcase (Portable)

Bootstrap 5.3 の新機能（ダークモード、拡張されたCSS変数など）を含め、主要な全機能を視覚的・インタラクティブに網羅したデモアプリケーションです。

npmやローカルサーバーの起動が不要で、**`index.html` をブラウザで直接ダブルクリックして開くだけで**すべての機能が動作するポータブル構成になっています。

---

## 🚀 使い方

1. 本フォルダにある `index.html` をブラウザ（Chrome, Safari, Edge, Firefoxなど）で直接開きます（ダブルクリックまたはドラッグ＆ドロップ）。
2. ブラウザのアドレスバーが `file:///.../index.html` となっている状態でも、すべての機能（ダークモード切り替え、JSプラグイン、CSS変数のリアルタイム変更など）が正常に動作します。

---

## 🎨 網羅しているBootstrap 5.3の機能

### 1. カラーモード (Color Modes)
- `data-bs-theme="light|dark"` の切り替え。
- OSの設定と連動する「オート」モードのサポート。
- 5.3で追加された新しいセマンティックカラー（`*-subtle` 背景、`text-*-emphasis` など）の表示デモ。

### 2. CSS変数 (CSS Variables) の動的変更
- スライダーを使用して、プライマリのテーマ色（色相）や要素の角丸サイズ（`border-radius`）をリアルタイムに変更可能。

### 3. グリッドとレイアウト
- レスポンシブグリッドシステム（ブレイクポイント別のレイアウト変化の可視化）。
- フレックスボックスによる垂直整列（`align-items-*`）や、隙間（Gutters, `g-*`）のプレビュー。

### 4. フォームとバリデーション
- 入力欄にラベルが被さるフローティングラベル（`floating labels`）。
- スイッチ、カラーピッカー、ファイル選択、レンジスライダー。
- 送信時にBootstrapのスタイルが適用されるクライアントサイド・バリデーション。

### 5. 各種コンポーネント (HTMLのみ)
- アコーディオン、アラート（警告）、バッジ、リストグループ、進行状況バー（Progress）、スケルトン表示（Placeholder）、スピナーなど。

### 6. インタラクティブなJavaScriptプラグイン
- **Carousel**: 自動再生・フェード切り替え付きのスライドショー。
- **Modal / Offcanvas**: 中央ポップアップと画面端からのスライドインメニュー。
- **Popover / Tooltip**: ホバー・クリック時の情報吹き出し。
- **Toast**: 画面右上に動的に通知を生成して表示するトースター機能。
- **Scrollspy**: 右側のスクロール位置に合わせて左側メニューのハイライトが自動追従。

### 7. ヘルパーとユーティリティ
- アスペクト比維持（`Ratio` による16:9動画埋め込み）。
- 親要素全体をクリック可能にする `Stretched Link`。
- テキストの1行自動省略（`Text Truncation`）。

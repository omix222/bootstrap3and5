Bootstrap 移行学習ガイド

Bootstrap 3 → Bootstrap 5.3

jQuery不要・グリッド刷新・クラス名変更など、主要な変更点を左右比較で学べます。

このガイドの使い方： 各レッスンをクリックすると、Bootstrap 3 のコードと Bootstrap 5 のコードを左右に並べて比較できます。 実際に動作するサンプルでクラス名の違いを確認しましょう。
01

グリッドシステム

col-xs 廃止・新ブレークポイント（xxl）・オフセット・Auto-layout Columns

col-xs → col offset xxl追加
02

ナビゲーションバー

data-toggle → data-bs-toggle・navbar-default 廃止・カラーモード

data-bs-* navbar-expand offcanvas
03

カード（panel / well / thumbnail 廃止）

BS3 の panel・well・thumbnail がすべて card に統合された

panel → card well → card thumbnail → card
04

フォーム

form-group 廃止・mb-3 へ・フローティングラベル・バリデーションスタイル

form-group廃止 floating label form-control
05

JavaScript / jQuery 不要化

Modal・Collapse・Tooltip が jQuery なしで動作する仕組みと data-bs-* 属性

jQuery不要 data-bs-toggle Modal API
Bootstrap 3 → 5 主要変更点チートシート
カテゴリ Bootstrap 3 Bootstrap 5
最小ブレークポイント col-xs-* col-*
JS トグル属性 data-toggle data-bs-toggle
JS ターゲット属性 data-target data-bs-target
依存ライブラリ jQuery 必須 Vanilla JS のみ
パネル系 panel / well / thumbnail card
フォームグループ form-group mb-3（廃止）
ナビバー色 navbar-default navbar-light / navbar-dark
最大ブレークポイント lg まで xxl まで追加
グリッド基盤 Float ベース Flexbox ベース
RTL サポート なし あり

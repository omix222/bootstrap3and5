Bootstrap 3
BS3 の問題点： data-toggle / data-target で制御。 navbar-default（灰色）または navbar-inverse（黒）の2択。 jQuery が必須。

Bootstrap 5
BS5 の変更点： data-bs-toggle / data-bs-target に統一。 navbar-default は廃止。navbar-expand-{breakpoint} でレスポンシブを制御。 jQuery 不要。

1. 基本的なナビバー（BS5）

MySite
  • ホーム
  • サービス
  • お問い合わせ
<nav class="navbar navbar-expand-lg navbar-light bg-light"> <div class="container-fluid"> <a class="navbar-brand" href="#">MySite</a> <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navBS5Demo"> <span class="navbar-toggler-icon"></span> </button> <div class="collapse navbar-collapse" id="navBS5Demo"> <ul class="navbar-nav"> <li class="nav-item"> <a class="nav-link active" href="#">ホーム</a> </li> </ul> </div> </div> </nav>

2. カラーバリエーション（多彩）

bg-primary
